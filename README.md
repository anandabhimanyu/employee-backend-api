# Employee Backend API

A production-ready backend API built using **Go**, **Gin**, and **PostgreSQL**.

This project was developed as part of a backend engineering take-home assignment to demonstrate API design, authentication, authorization, database modeling, and production-quality engineering practices.

---

##  Features

- JWT-based authentication
- Role-based authorization (RBAC)
- Employee CRUD APIs
- Pagination, filtering, and sorting
- PostgreSQL persistence
- Input validation using DTOs
- Clean project structure
- Docker-ready setup
